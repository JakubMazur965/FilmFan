## General info
FilmFun is a simple GUI movie portal. It can be used for searching and rating movies.

## Features
- Signing Up 
- Signing In
- Logging Out



- Top 3 movies
- Searching for movie which interests you



- Rating the movie
- Displaying all movie's details

- Serializable Interface 
- GUI
